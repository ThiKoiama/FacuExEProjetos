Introdução a Web

Introdução à Web é uma série concisa que apresenta os aspectos práticos do desenvolvimento da web. Você configurará as ferramentas necessárias para construir uma página da web simples e publicar seu próprio código simples.

Quais ferramentas eu realmente preciso, agora?

Parece uma lista assustadora, mas, felizmente, você pode começar no desenvolvimento da web sem saber nada sobre a maioria deles. Neste artigo, vamos apenas configurá-lo com o mínimo: um editor de texto e alguns navegadores modernos.

Instalando um editor de texto :

Você provavelmente já tem um editor de texto básico em seu computador. Por padrão, o Windows inclui o Bloco de notas e o macOS vem com o TextEdit. As distros Linux variam; O Ubuntu vem com o gedit por padrão. Para desenvolvimento web, você provavelmente pode fazer melhor do que o Bloco de Notas ou o TextEdit. Recomendamos começar com o Visual Studio Code, que é um editor gratuito que oferece visualizações ao vivo e dicas de código.
